# Liga Fácil - Backend (Spring Boot)

API para gerenciamento de campeonatos de futebol, parte do projeto fullstack **Liga Fácil**.  
Projeto voltado para portfólio de estágio Fullstack Java, com foco em boas práticas de backend e preparação para integração com frontend.

---

## Tecnologias
- Java 21
- Spring Boot 4.0.3
- PostgreSQL
- JUnit / Mockito para testes unitários automatizados

---

## Funcionalidades principais
- Criar e listar campeonatos
- Adicionar e editar clubes em um campeonato
- Geração automática de partidas (Round Robin)
- Consulta de partidas por rodada
- Classificação dinâmica dos campeonatos
- API REST preparada para integração com frontend

---

## Status do projeto
- Backend funcional e pronto para consumo pelo frontend
- Arquitetura em camadas (Controller → Service → Repository → Database)
- MVP implementado: campeonatos, times, partidas e classificação automática
